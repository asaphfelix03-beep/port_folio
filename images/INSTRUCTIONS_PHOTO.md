# Instructions pour ajouter votre photo de profil

## Étapes à suivre :

1. **Trouvez votre photo** sur votre ordinateur
2. **Copiez-la** dans ce dossier (`images/`)
3. **Renommez-la** en `profile-photo.jpg`

## Formats acceptés :
- JPG/JPEG (recommandé)
- PNG
- WebP

## Taille recommandée :
- Minimum : 300x300 pixels
- Idéal : 400x400 pixels ou plus
- Format carré (1:1) pour un meilleur rendu

## Nom du fichier :
Le fichier doit s'appeler exactement : `profile-photo.jpg`

## Vérification :
Une fois la photo ajoutée, ouvrez `index.html` dans votre navigateur pour voir votre photo dans la section "À propos".
