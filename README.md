# Portfolio Moderne - HTML/CSS/JavaScript

Un site portfolio moderne, élégant et professionnel créé avec HTML, CSS et JavaScript vanilla.

## 🎨 Caractéristiques

- **Design moderne** avec palette de couleurs harmonieuse (tons foncés + accents bleus/turquoises)
- **Responsive design** qui s'adapte parfaitement aux écrans mobiles et desktop
- **Animations fluides** : fade-in au scroll, effets de survol, transitions douces
- **Navigation sticky** avec menu mobile hamburger
- **5 sections principales** : Accueil, À propos, Projets, Compétences, Contact
- **Typographie moderne** (Poppins) avec tailles adaptatives
- **Code commenté** et bien structuré
- **Performance optimisée** sans frameworks lourds

## 🎯 Sections détaillées

1. **Accueil** : Message de bienvenue avec CTA animé
2. **À propos** : Présentation avec photo placeholder
3. **Projets** : 3 exemples de projets avec technologies
4. **Compétences** : Organisées par catégories (Frontend, Backend, Outils)
5. **Contact** : Email et liens réseaux sociaux
6. **Footer** : Simple avec droits d'auteur

## 🚀 Utilisation

Ouvrez simplement `index.html` dans votre navigateur pour voir le site en action. Tout est prêt à être personnalisé avec vos informations !

## 📝 Personnalisation rapide

1. **Votre nom** : Remplacez "Votre Nom" dans `index.html`
2. **Email** : Mettez à jour `votre.email@example.com`
3. **Liens sociaux** : Modifiez les URLs LinkedIn et GitHub
4. **Projets** : Ajoutez vos propres projets dans la section Projets
5. **Compétences** : Personnalisez selon votre profil

## 🎨 Couleurs personnalisables

Les couleurs principales sont définies dans `:root` dans `styles.css` :
- `--bg-primary`: #0a0a0a (fond principal)
- `--accent-primary`: #00d4ff (bleu principal)
- `--accent-secondary`: #00ff88 (vert accent)

- Espacement optimisé pour petits écrans
